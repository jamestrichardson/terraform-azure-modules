# terraform-azure-modules

This repo hosts several, reusable Terraform modules intended to build different infrastructure in Azure.

## Repository Structure

```
terraform-azure-modules/
├── modules/      # Reusable Terraform modules
├── examples/     # Example configurations showing module usage
└── docs/         # Additional documentation
```

## Modules

See the [`modules/`](./modules/) directory for all available modules and their documentation.

## Examples

See the [`examples/`](./examples/) directory for end-to-end usage examples.
