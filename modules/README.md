# Modules

This directory contains reusable Terraform modules for provisioning Azure infrastructure.

## Structure

Each module lives in its own subdirectory and follows this layout:

```
modules/
└── <module-name>/
    ├── main.tf
    ├── variables.tf
    ├── outputs.tf
    ├── versions.tf
    └── README.md
```
